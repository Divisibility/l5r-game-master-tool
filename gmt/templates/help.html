Help

Generator

  • Generate will create a character with the select states. Thereby 'Random' states will be choose by the generator, everything else will be used as is. If any combination of states are impossible, the generator will put the states in question back to 'Random'.
  • Reset will reset all states to 'Random'.
  • Save will save all states if everything is correct and will report any problems.

Manager

  • Load will show the selected character with all available states.
  • Edit will let you write some notes for the selected character.
  • Save will save 'Notes' to the selected character.
  • Delete will delete the selected character permanently.
